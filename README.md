# Simple BMI Calculator Web Application
This is a simple web-based BMI calculator application that allows users to calculate their BMI based on their weight and height. The application is built using HTML, CSS, and PHP.

Features:

-Simple and easy to use
-Calculates BMI instantly
-Displays the result in a clear and easy-to-understand format

How to Use

-Clone or download the repo to your local machine.
-Open the index.php file in your web browser.
-Input your weight and height in the form provided.
-Click the "Calculate" button to calculate your BMI.
-Your BMI result will be displayed on the screen.
